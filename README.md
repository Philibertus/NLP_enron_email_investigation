# NLP Python Project

This is a starter workspace for Natural Language Processing (NLP) tasks using Python.

## Structure
- Place your source code in the `src/` directory (create it if needed).
- Add notebooks in the `notebooks/` directory (create it if needed).
- Add data files in the `data/` directory (create it if needed).

## Setup
- Python 3.13.5 (system environment)
- Recommended VS Code extensions: Python, Python Environment Manager

## Getting Started
1. Install dependencies (if any):
   ```zsh
   # Example: pip install -r requirements.txt
   ```
2. Add your code in `src/` or notebooks in `notebooks/`.
3. Run your scripts:
   ```zsh
   /usr/local/bin/python3 your_script.py
   ```
